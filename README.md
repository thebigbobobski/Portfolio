#  Online Retail Sales

## About Data
This data set displays retail sales for a UK based online shop.
This data set contains a csv file with 541909 records, one for each transaction.


## Data Cleaning and pre-processing
- Converted the Customer ID column from a float to a string.
- Converted the Invoice Date to Date/Time instead of integer.
- Dropped rows that were missing Customer ID.
- Dropped rows that were missing descriptions.

- Retrieved cancelled orders, to get exact figure.
- Then removed all cancelled orders for revenue analysis.

## Revenue Analysis
- Created a revenue column.
- Then created a customer summary table,  which included first and last purchase date, total orders/revenue, average order value and total items purchased.
